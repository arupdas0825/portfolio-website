# Arup Das — High-Fidelity Portfolio

A state-of-the-art digital showcase built with React, featuring a premium dark-themed aesthetic, high-end 3D interactions, and cinematic storytelling through photography.

## 🎨 Design Philosophy
This portfolio is crafted for visual excellence and fluid user experience. It utilizes **Glassmorphism**, **Ambient Glow Effects**, and **Cinematic Typography** to create a premium digital environment. The design prioritizes content hierarchy and smooth micro-interactions to keep users engaged from hero to footer.

## 🚀 Key Features

### 1. Interactive 3D Orbital Widget
A custom-engineered Hero component using **Three.js** and **CSS 3D Transforms**. It features a central pulse-core surrounded by rotating rings and floating tech-stack icons that maintain their orientation via counter-rotation algorithms.

### 2. Cinematic Photography Gallery
A dedicated space for visual storytelling. 
- **Immersive Lightbox**: A custom-built split-view modal that pairs high-resolution imagery with narrative descriptions.
- **Smart Navigation**: Seamless transition between photos using on-screen controls or keyboard shortcuts.
- **Dynamic Loading**: Optimized image handling with fallback states for missing assets.

### 3. High-End Scrolling & Navigation
- **Lenis Smooth Scroll**: Buttery smooth, momentum-based scrolling for a desktop-app feel.
- **Scroll Spy**: A smart navbar that tracks the user's position and highlights active sections in real-time.
- **Hash Navigation**: Zero-latency smooth transitions between local sections (Home, About, Work, Photography, etc.).

### 4. Responsive & Fluid Layout
Fully adaptive architecture that transforms from a desktop-optimized floating navbar to a thumb-friendly bottom navigation system on mobile devices.

## 📁 Project Structure
- `src/Home.js` — Hero section with 3D widget logic.
- `src/Gallery.js` — Photography engine and lightbox system.
- `src/About.js` — Professional profile and skill taxonomy.
- `src/App.css` — Core design system and glassmorphism utilities.
- `src/Navbar.js` — Scroll-spy navigation controller.
